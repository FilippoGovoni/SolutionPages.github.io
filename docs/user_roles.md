<h1>Customer Portal</h1>
<p>
<code>Updated: 19/03/2019</code><br>
The Customer Portal role provides, to customers of the service, visibility and access to Fastservice.
This role has visibility of forms and features showed below.
</p>

<h2>Navbar</h2>
<img width="1040" alt="customer portal navbar" src="https://user-images.githubusercontent.com/2674665/53348962-5df95680-391c-11e9-826d-a9751a6f0b49.png">
<p>
For the Customer portal role are active these menus:
<ul>
<li>My desk. My Desk menu is active, with these submenus:</li>
<ul>
<li>Ticket list.</li>
<li>Dashboard.</li>
</ul>
<li>User User menu is active, with these submenus:</li>
<ul>
<li>User profile.</li>
<li>Change password.</li>
<li>Logout.</li>
</ul>
</ul>
</p>
<h2>Ticket list</h2>
<p>
A customer portal can create tickets as well as via the ticket by mail function, also directly from the Fastservice web portal using the New ticket button
</p>
<img width="1299" alt="Schermata 2019-03-19 alle 17 25 21" src="https://user-images.githubusercontent.com/2674665/54623552-073ef280-4a6c-11e9-8456-3c65313a3d1a.png">

<h2>Ticket edit</h2>
<p>
Ticket modification is completely disabled for a user with a customer portal role.
The user can see the contents of the ticket fields but cannot modify them.
However, to facilitate ticket resolution, the File tab and the Messages tab are enabled to allow the customer portal user to add files and insert messages.
</p>
